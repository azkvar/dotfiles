# dotfiles

Personal dotfiles for macOS.

## Install

```bash
git clone git@github.com:sumiyuki-sub/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./install.sh
```

## Link

```bash
./link.sh
```

Existing files are backed up with a `.bak` suffix before linking.

## Unlink

```bash
./unlink.sh
```

## .zshrc運用

- 共通設定 `~/.zshrc`
- 個々の設定 `~/.zshrc.local`
